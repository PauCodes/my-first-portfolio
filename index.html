Let's Get Quizzical!

React | HTML5 | CSS3 | API | Firebase

No plans for Friday night? No problem! Let's Get Quizzical is an interactive trivia game that allows the user to select a category as well as the number of questions they want to play. Using React and Firebase, this app stores all past games on Realtime Database, so that the user can replay old games any time. This project was created with three other great developers: Joey Sea, Lester Yiu and Mel Peto

Marvel Battle

React | HTML5 | CSS3 | API

Are you a Marvel fan? I'm not! But if your answer is yes, this app is for you!This ultimate Marvel Battle app allows the user to pick their favourite Avengers Character or just input the name of a random superheroe. US Park Finder

HTML5 | SASS | Javascript | API

Planning a trip to one of your favourite national park in America? This US Park Finder can help you! This app was built using vanilla Javascript and grabs data from the NPS National Parks API based on user selection of state and preferred outdoor activity. This project was developed together with another awesome developer Natalie Laing
